# Solid Propellant Rocket Motors: Internal Ballistics

An open technical resource covering the internal ballistics of solid propellant rocket motors.

## Read the Book

Online Version:
https://altustd.github.io/solid-rocket-internal-ballistics/

Local Version:

cd solid-rocket-internal-ballistics
quarto render
