# Solid Propellant Rocket Motors: Internal Ballistics

An open technical resource covering the **internal ballistics** of solid propellant rocket motors (SRMs). 

This ebook explores propellant chemistry, grain geometry, burning surface evolution, chamber pressure prediction, thrust curves, and overall motor performance — designed for students, experimental rocketeers, and amateur propulsion enthusiasts.
