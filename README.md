# Solid Propellant Rocket Motors: Internal Ballistics

An open technical resource on the internal ballistics of solid propellant rocket motors. This book covers the theory, equations, and computational methods used in solid rocket motor design and analysis.

## Read the Book

View the Full Book: https://altustd.github.io/solid-rocket-internal-ballistics/

## Table of Contents

1. Introduction — Overview of internal ballistics
2. Solid Propellants — Chemistry, classification, and properties
3. Grain Design and Geometry — Web thickness, burning surface evolution, common geometries
4. Internal Ballistics Core — Mass balance, pressure prediction, and thrust generation

More chapters on burning rate laws, transients, numerical modeling, and applications are in development.

## What This Book Covers

- Propellant combustion and burning rate laws
- Solid grain geometry and design principles
- Chamber pressure and mass flow balance
- Thrust curve generation and performance analysis
- Numerical simulation techniques

## Project Goals

- Provide clear technical content for amateur and experimental rocketeers
- Bridge theory with practical motor design
- Include equations, diagrams, and eventually simulation code
- Serve as a learning resource for students and hobbyists
